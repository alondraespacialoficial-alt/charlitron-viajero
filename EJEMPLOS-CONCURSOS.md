# 🎯 EJEMPLOS DE CONCURSOS - IDEAS Y PLANTILLAS

## 1️⃣ "LA FOTO MÁS ANTIGUA"

**Pregunta:** "¿En qué década fue tomada esta foto?"

**Opciones:**
- ❌ 1950s-1960s
- ✅ 1920s-1930s (CORRECTA)
- ❌ 1890s-1900s

**Por qué funciona:**
- Educativo + visual
- Genera engagement sobre historia
- Fácil de compartir: "Adiviné la década de la foto más antigua 🏆"

**Propina de contenido:**
- Usa foto de tu familia
- O compra en Unsplash histórica
- El misterio = atrae clicks

## 2️⃣ "OFICIOS PERDIDOS"

## 🎬 PLANTILLA RÁPIDA

```
1. ELIGE TEMA
   ├─ Familia
   ├─ Región/Viajes
   ├─ Época histórica
   └─ Objeto/Tradición

2. ENCUENTRA IMAGEN
   ├─ Tu archivo familiar
   ├─ Periódico/revista
   ├─ Unsplash.com (historias)
   └─ Pixabay.com (vintage)

3. ESCRIBE PREGUNTA
   ├─ Simple (1 frase)
   ├─ Intrigante (genera curiosidad)
   └─ Educativa (enseña algo)

4. CREA 3 OPCIONES
   ├─ 1 CORRECTA ✅
   ├─ 1 cercana ❌
   └─ 1 obvia falsa ❌

5. AGREGA CONTEXTO (opcional)
   ├─ Nombre ganador
   ├─ Descripción
   └─ Tema relacionado

6. GUARDA Y PUBLICA
```

## 📊 ESTRATEGIA DE LANZAMIENTO

**Mes 1:** 2 concursos (pruebas)
```
  Semana 1: "Foto más antigua"

```

```

  Tema: Rota entre familia/historia/viajes
```

```
  Semanal + Especiales (festividades)
  Semanal + Sponsorizado (marcas)
```

## 🎨 TIPS PARA VIRALIDAD

**¿Cuándo publicar?**
- Viernes PM (fin de semana)
- Martes por la mañana (coffee break)
- Jueves 6-8 PM (después del trabajo)

**¿Cómo escribir el caption?**
```
"¿Sabes la respuesta? ¿Gánate un código exclusivo! 🔥
Comparte en comentarios SOLO SI ACIERTAS 👀"
```

**¿Dónde repostearlo?**
- Instagram Stories (swipe-up si tienes)
- TikTok (trend: "adivina la era")
- Facebook (personas mayores aman historia)
- Twitter/X (engagement rápido)

**¿Qué pasa con el ganador?**
```
Opción 1: Manual
  "El primero que lo comparta es ganador"
  
Opción 2: Sorteo
  "Ganador sorteado entre todos los correctos"
  
Opción 3: Automático
  "Quien comparta primero tiene el código"
```
